    AI Transparency Media Lab - v0.1.0

    AI Transparency Media Lab

    Веб-застосунок для інтелектуального аналізу медіаконтенту з анімованою візуалізацією результатів розпізнавання об'єктів та тексту.

    Про проєкт

    Дипломна робота на тему "Інформаційна система інтелектуального аналізу медіаконтенту з анімованою візуалізацією результатів розпізнавання об'єктів".

    Система працює повністю на стороні клієнта — всі AI-моделі завантажуються та виконуються безпосередньо у браузері без передачі даних на сервер.

    Модулі

    OCR Lab

    Вилучення друкованого тексту з графічних файлів за допомогою Tesseract.js. Підтримка української та англійської мов. Анімація лазерного сканування та виділення розпізнаних блоків тексту на зображенні.

    Smart Gallery

    Автоматична класифікація та каталогізація фотографій через MobileNet v2. Визначення категорій: люди, тварини, техніка, меблі, природа та інші. Анімоване групування карток по категоріях.

    Object Detection

    Пошук та ідентифікація об'єктів у зображенні або відеопотоці (веб-камера) в режимі реального часу через COCO-SSD. Відображення confidence score для кожного виявленого об'єкта.

    Media Transcriber & Sentiment

    Розпізнавання мовлення через Web Speech API та аналіз тональності тексту. Колірна індикація позитивних, негативних та нейтральних фрагментів.

    Технічний стек

    Технологія Призначення
    React 18 + Vite + TypeScript Frontend фреймворк
    Tesseract.js OCR розпізнавання тексту
    TensorFlow.js Платформа для AI моделей
    MobileNet v2 Класифікація зображень
    COCO-SSD Детекція об'єктів
    GSAP 3 Анімації інтерфейсу
    React Router DOM v6 Маршрутизація

    Запуск проєкту

    # Встановити залежності
    npm install

    # Запустити в режимі розробки
    npm run dev

    # Зібрати для продакшну
    npm run build

    Вимоги

    • Node.js 18+
    • Сучасний браузер (Chrome, Firefox, Safari, Edge)
    • Для модуля веб-камери — дозвіл на доступ до камери
    • Для модуля транскрибації — дозвіл на доступ до мікрофону
