# Генерація документації

## Інструмент

**TypeDoc** — генератор документації для TypeScript проєктів.
Автоматично створює HTML документацію з JSDoc коментарів у коді.

## Встановлення
```bash
npm install --save-dev typedoc
```

## Генерація документації
```bash
npm run docs
```

Документація генерується у папку `docs/api/`.

## Конфігурація

Файл `typedoc.json` у корені проєкту:

- `entryPoints` — вхідні файли для аналізу
- `out` — папка для згенерованої документації
- `name` — назва проєкту у документації
- `excludePrivate` — включати приватні члени

## Стандарти документування

Використовується формат **JSDoc**. Кожна публічна функція має містити:

- `@description` — детальний опис
- `@param` — опис параметрів
- `@returns` — опис значення що повертається
- `@example` — приклад використання

### Приклад
```typescript
/**
 * Короткий опис функції.
 *
 * @description Детальний опис логіки.
 * @param {string} label - Опис параметра
 * @returns {string} Опис результату
 * @example
 * myFunction('test') // повертає 'result'
 */
function myFunction(label: string): string {
```

## Автоматична публікація

Документація публікується на GitHub Pages автоматично
через GitHub Actions при кожному push у гілку `master`.
